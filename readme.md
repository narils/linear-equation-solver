# Linear Equation Solver
Run main with command line arguments
`-in input.xt` and `-out output.txt`.

## Input format
Textfile with LF
a = number of unknowns
b = number of equations (can be ommitted)
```text
a b
x1 y1 z1 c1
x2 y2 z2 c2
x3 y3 z3 c3
```


###Example input
```text
3 4
0 1 2 9
0 1 3 1
1 0 6 0
2 0 2 0
```

3 unknowns with 4 equations.
